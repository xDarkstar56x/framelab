# FrameLab - Estudio

FrameLab es una herramienta web de código abierto diseñada para fotógrafos que buscan automatizar la presentación editorial de sus imágenes. Extrae instantáneamente los metadatos técnicos (EXIF) y genera marcos personalizables de alta resolución listos para publicar.

## 🚀 Características Principales

*   **Privacidad Total (Client-Side):** Todo el procesamiento de la imagen y extracción de metadatos ocurre localmente en el navegador. Las fotografías nunca se suben a ningún servidor.
*   **Lectura Automática de EXIF:** Detecta cámara, lente, apertura, velocidad de obturación, ISO, distancia focal, fecha y coordenadas GPS.
*   **Extracción de Paleta de Colores:** Genera automáticamente una paleta de colores predominantes en la imagen para complementar el diseño.
*   **Motor Tipográfico Inteligente:** Ajuste automático (Responsive) para evitar que los textos largos se solapen, apilando la información dinámicamente.
*   **Exportación en Alta Calidad:** Soporta formatos JPEG y PNG, manteniendo escalas de resolución exactas a la imagen original.

## 🛠️ Uso

1. Selecciona una imagen desde tu dispositivo.
2. Escoge un "Estilo Base" para configurar rápidamente los márgenes y tipografías.
3. Personaliza el encuadre (Crop), los colores de fondo y texto, y la visibilidad de la sombra 3D.
4. Exporta tu imagen con los parámetros de compresión que necesites.
